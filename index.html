How to Use the Serial Console

Getting Started

  1. Use a Compatible Browser: This tool requires the Web Serial API, available in browsers like Google Chrome and Microsoft Edge.
  2. Connect Your Devkit: Physically connect your SiMa.ai Devkit to your computer using a USB cable.

Connecting to Devices

  1. Click the [+] New Connection button in the command palette.
  2. A browser pop-up will appear. Select the serial port for your Devkit and click "Connect". A new tab will be created for this connection.
  3. The console will automatically attempt to log you in. You can repeat this process to connect to multiple devices.

Using the Interface

Interact with the active tab:

  • Directly in the Terminal: Type any command into the terminal window and press Enter.
  • Command Palette: All buttons apply to the currently selected terminal tab.

Disconnecting

Click the (x) on a tab to log out and close that specific connection.
